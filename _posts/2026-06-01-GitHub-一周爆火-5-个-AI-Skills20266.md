# 🔥 GitHub 一周爆火 5 个 AI Skills（2026.6）

## 1️⃣ last30days-skill（+12.6k⭐）

全网热点雷达：同时爬取 Reddit、YouTube、Hacker News 等平台，整理近 30 天主题热度、观点与反馈，输出带来源的整合报告，告别多页调研。

```bash
npx skills add mvanhorn/last30days-skill
```

📎 [github.com/mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill)
