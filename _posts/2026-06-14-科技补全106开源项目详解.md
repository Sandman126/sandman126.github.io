## ① WechatOnCloud — Docker 云微信

**一句话**：在 Docker 上跑微信服务端，多设备消息同步、多实例管理。

**核心功能**：
- 在飞牛 NAS（x86_64 / arm64）或任意 Docker 主机上运行服务端微信
- 可管理**多个微信实例**，每个实例是独立的微信会话
- 多个 Web 用户通过浏览器访问被授权的实例
- 实现**跨设备消息同步、多端共享**
- **不修改微信客户端**，纯容器化方案

**相关项目**：`docker-wechat`（RICwang/docker-wechat）功能类似——在 Docker 中运行微信客户端，通过 Web（端口 5800）或 VNC（端口 5900）访问，支持深色模式、音频播放等。

**链接**：
- WechatOnCloud：[https://github.com/Gloridust/WechatOnCloud](https://github.com/Gloridust/WechatOnCloud)
- docker-wechat：[https://github.com/RICwang/docker-wechat](https://github.com/RICwang/docker-wechat)

## ② plain-app — 浏览器管理 Android 手机

**一句话**：在手机上启动 Web 服务器，同一局域网下任何设备的浏览器都能管理手机。

**核心功能**：
- 🔒 **隐私安全** — 纯本地局域网，无云端、无账号、无数据上传
- 📁 **全功能文件管理** — 内置存储、SD 卡、U 盘全目录浏览，支持上传/下载/重命名/移动/删除
- 💬 **消息通讯** — 实时同步手机通知、短信、通话记录、联系人，浏览器可直接发短信
- 🎬 **多媒体流媒体** — 图片/音频/视频在线预览播放，无需下载
- 📱 **应用管理** — 浏览器中列出已安装应用、导出 APK
- 🛠️ **内置工具** — Markdown 笔记、RSS 阅读器、P2P 文件传输、番茄时钟、噪音检测
- 🖥️ **屏幕镜像** — 通过 WebRTC 实时投射手机屏幕到浏览器（最高 30fps + 音频）

**使用方式**：安装 APK → 打开 App 点击启动 → 同一 Wi-Fi 下用浏览器访问显示的地址即可。

**链接**：
- GitHub：[https://github.com/plainhub/plain-app](https://github.com/plainhub/plain-app)
- 官网：[https://plainapp.app/zh](https://plainapp.app/zh)
- Google Play：[https://play.google.com/store/apps/details?id=com.ismartcoding.plain](https://play.google.com/store/apps/details?id=com.ismartcoding.plain)
- F-Droid：[https://f-droid.org/packages/com.ismartcoding.plain/](https://f-droid.org/packages/com.ismartcoding.plain/)

## ③ Pinkbin — AI 辅助磁盘清理工具

**一句话**：开源 Windows 磁盘清理工具，AI 帮你判断陌生文件夹能不能删。

**核心功能**：
- ⚡ **秒级扫描** — Windows 上直读 NTFS Master File Table，2-5 秒扫完整盘 C 盘空间分布
- 🤖 **AI 分析** — 看不懂的文件夹拖拽到聊天框，AI 解释「这是什么」「能不能删」「删了丢什么」
- 🧹 **应用专属清理** — 常见大软件（微信缓存、Conda 环境等）有独立清理脚本
- 🛡️ **安全机制** — 默认进回收站，**永不读取你的文件内容**
- 🔑 **灵活 AI 后端** — 支持 Anthropic / OpenAI / Gemini API Key，或本地 Ollama 完全免费

**链接**：
- GitHub：[https://github.com/cccyd2003-qwq/pinkbin](https://github.com/cccyd2003-qwq/pinkbin)

## ④ gaokaomath — 历年高考数学真题集

**一句话**：收集历届高考数学试卷的开源数据集，用于学习参考和大模型推理评测。

### 三个相关项目

**deekur/gaokaomath** — 传统试卷归档
- 收集从 **1952 年至今** 的高考数学真题
- 涵盖全国卷、各省自主命题卷、新课标卷、新高考卷
- 按年份和省份分类，附带高考改革史
- ⭐ 442 Stars
- 链接：[https://github.com/deekur/gaokaomath](https://github.com/deekur/gaokaomath)

**FrankieYao/GaoKaoMath** — LLM 评测 JSON 数据集
- 结构化 JSON 格式（题目、答案、分值、年份、省份）
- MIT 开源许可，持续更新
- 发布在 Hugging Face | 设计初衷：大模型数学推理评测基准
- 链接：[https://huggingface.co/datasets/FrankieYao/GaoKaoMath](https://huggingface.co/datasets/FrankieYao/GaoKaoMath)

**ranxi2001/freshbench** — 2026 高考数学时间戳基准
- 2026 年高考数学刚出题就做成可复现评测基准
- 包含 19 道题（单选、多选、填空、解答、计算、证明）
- 每条题目标注采集时间与核验状态，防止数据污染
- 链接：[https://github.com/ranxi2001/freshbench](https://github.com/ranxi2001/freshbench)
